## Terminal tree view

Run `prefablens main` in a Unity project to compare a ref against your working
tree. Changed files come back as a GameObject / component / field tree instead
of a wall of YAML, and guid references — scripts, materials, source prefabs —
resolve to their asset paths automatically.
