# Demo fixtures

Before/after snapshots of the files changed in
[unity-yaml-playground#2](https://github.com/hashiiiii/unity-yaml-playground/pull/2)
(base `7682a58`, head `d7cf9b9`), generated in Unity 6000.5.2f1 by that repo's
FixtureGenerator. `before/` is the base state, `after/` the head state; added,
removed, and renamed files exist on one side only.

`build.mjs` turns these trees into a throwaway git repository so the demo pages
show real `prefablens` output for a realistically sized pull request.
